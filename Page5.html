Où nous écouter ?

Vous pouvez écouter nos podcasts ici, ici, et ici.

Sur votre ordinateur: vous pouvez écouter directement sur le site d'un producteur de podcasts (Louie, Radio France, Nouvelles Écoutes, Binge Audio, Boxsons...)

Vous pouvez aussi nous écouter sur des plateforms qui agrègent les podcasts les podcasts de nombreux producteurs, comme Soundcloud, ou YouTube. Si vous avez iTunes sur votre ordinateur, vous pouvez encore vous abonner directement depuis l'iTunes Store.

Sur mobile: si vous avez un iPhone, vous avez par défaut l'application Apple Podcasts. Dedans vous pouvez chercher le nom du podcast que vous souhaitez écouter.

  • Si vous êtes sur Androïd, vous pouvez écouter sur Podcast Addict, Overcast, Stitcher...
  • Si vous avez un Windows Phone, cherchez l'application podcasts qui est par défaut dans votre téléphone.
